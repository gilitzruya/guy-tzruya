# Homepage “קצת עלי” portrait

Portrait images for the **About teaser on the homepage only** (`HomeAboutSection`).
The full About page (`/about`) uses separate files under `public/about/hero/`.

## Required files

| File | Scene | Notes |
|------|--------|--------|
| `portrait-day.webp` | Day mode | Homepage portrait |
| `portrait-night.webp` | Night mode | Optional separate grade; can match day file |

## Recommended specs

- **Format:** WebP (preferred)
- **Aspect ratio:** ~4:5 (portrait), or wider if the layout is split image + text
- **Dimensions:** ~1200×1600 px or larger
- **Subject:** Replace placeholders with the homepage-specific photo (e.g. Guy at the desk)

## Do not overwrite

Keep `public/about/hero/portrait-*.webp` for the dedicated About page.
