# About page assets

Portrait images for the **קצת עלי / About** hero section.

## Required files

| File | Scene | Notes |
|------|--------|--------|
| `hero/portrait-day.webp` | Day mode | Primary portrait |
| `hero/portrait-night.webp` | Night mode | Optional separate grade; can match day file |

## Recommended specs

- **Format:** WebP (preferred) or high-quality JPEG/PNG converted to WebP
- **Aspect ratio:** ~4:5 (portrait)
- **Dimensions:** ~1200×1600 px or larger
- **Subject:** Guy Tzruya, professional portrait, neutral background works well with the editorial frame

## Optional journey images

`journey/step-01.webp` … `step-05.webp` — small thumbnails for timeline steps (not required; timeline works without images).

## Fallback

If `portrait-day.webp` / `portrait-night.webp` are missing, the hero shows a styled **GT** placeholder until files are added.
